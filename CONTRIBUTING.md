# This is a fork of a public repo

| Command                             | Remote Used       | Meaning                                  |
| ----------------------------------- | ----------------- | ---------------------------------------- |
| `git pull` (on feature branch)      | `origin`          | Pull updates from **your fork**          |
| `git pull origin main`              | `origin`          | Pull updates from **your fork's main**   |
| `git fetch upstream-dexidp`         | `upstream-dexidp` | Fetch latest **public Dex repo** commits |
| `git merge upstream-dexidp/master`  | upstream-dexidp   | Merge public project’s master branch     |
| `git rebase upstream-dexidp/master` | upstream-dexidp   | Rebase on top of the public project      |
